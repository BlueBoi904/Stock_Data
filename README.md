# Stock Data API

Custom API and Client built with GO and Typescript

## Description

Our RESTful API is designed to consume realtime stock data using websockets.

### Installing and Executing

- To run our api

```
  cd api
  dep ensure
  go run main.go
```

- To run the client

```
  cd client
  yarn install
  yarn start
```
